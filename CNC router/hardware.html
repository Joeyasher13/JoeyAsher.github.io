It was decied for the project to use split phase 240 VAC for the router. It probably could have worked to run it on 120 VAC, but this gave me an me an excuse to get my parents to install a dedicated 50A 240VAC outlet in the garage. I am still trying to convince them to buy a welder, which is the second part to my master plan, but for now at least I have more than enough power for my CNC. The following hand-drawn schematic shows the basic power distribution for the system. Note that there are 4 breakers total, There is a 2 phase 40A breaker on the input, a 5A 2 phase breaker for the DC power supply, a 20A single phase breaker for the router, and finally a 20A single phase breaker to power a vacuum. This isn't the most detailed schematic, so some signals might be not be included, but it does give a good sense of the flow of power. AnAnTek 800W 56V DC power supplywas used for this router to power the stepper motors. In addition, this power supply has 24V and 5V rails that are used for powering all the other DC components in the system.

As mentioned earlier, it was crucial to create an organized enclosure for all the electronic components. This required the fabrication of a few custom parts. The first custom part was the panel for the Aviation connectors. I designed this component using Fusion 360 and had it 3D printed by Xometry using an HP MultiJet Fusion printer, resulting in a beautifully crafted piece. However, there is an unfortunate flaw in this design that I overlooked before sending it out for manufacturing. I positioned the screws to secure the panel on the inside of the box, intending to hide the screw heads. Regrettably, this design choice makes it nearly impossible to add new connections to the panel, as you can't access the inside without removing everything from the box.
